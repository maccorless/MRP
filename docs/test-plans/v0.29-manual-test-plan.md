## Regression Checklist

After completing the above, confirm no regressions in these core flows:

- [ ] Submit a new EoI application (English) — reaches NOC queue with status `pending`
- [ ] NOC admin approves an application — status changes to `approved`
- [ ] NOC admin returns an application — applicant can view returned status via magic link
- [ ] PbN: NOC allocates slots to an approved org — slot totals update correctly
- [ ] PbN: Submit PbN to OCOG — status changes to `noc_submitted`
- [ ] OCOG: Approve PbN — status changes to `ocog_approved`
- [ ] ENR: Add nomination, submit to IOC — list locks
- [ ] Audit trail: Actions above all appear with correct action types and actors
- [ ] IF admin path (if provisioned): Quota dashboard loads, allocation visible
